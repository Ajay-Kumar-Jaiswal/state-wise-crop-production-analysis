🌾 State-wise Analysis of Crop Production Trends in India

🚀 This project focuses on analyzing agricultural production data across different states, crops, and seasons in India, along with building a machine learning model to predict crop production.

🔗 **GitHub Repository:**  
https://github.com/Ajay-Kumar-Jaiswal/state-wise-crop-production-analysis

---

📊 Project Overview

Agriculture plays a vital role in India's economy. This project aims to:

- Analyze crop production trends across states 
- Understand relationships between area and production  
- Identify high-performing states and crops  
- Build a predictive model for crop production  

---

🔍 Key Features

- 📌 Data Cleaning & Preprocessing  
- 📈 Exploratory Data Analysis (EDA)  
- 🌍 State-wise Production Analysis  
- 🌾 Crop-wise Distribution Insights  
- ⚡ Yield (Efficiency) Analysis  
- 🔥 Heatmap for State vs Crop comparison  
- 🤖 Machine Learning Model for prediction  

---

📊 Visualizations

The project includes multiple visualizations such as:

- Top States by Production  
- Year-wise Production Trend  
- Area vs Production Relationship  
- Crop-wise Distribution  
- Yield Analysis  
- Heatmaps for Crop-State patterns  

---

🤖 Machine Learning Model

- Model Used: **Random Forest Regressor**  
- Target Variable: Crop Production  

 📈 Model Performance

- **R² Score:** ~0.91  
- Strong predictive performance on test data
